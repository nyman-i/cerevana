RRT — the Syllogimous lineage

Cerevana's RRT is built directly on Syllogimous: created by 4skinSkywalker, developed into Syllogimous-v3 by ikokusovereignty, and forked and refined by soamsy — soamsy/Syllogimous-v3 (playable original) is the codebase Cerevana grew from. giladkingsley is also a credited Syllogimous contributor.

N-Back — Brain Workshop

The dual n-back protocol Cerevana reimplements comes from Brain Workshop by Paul Hoskinson with Jonathan Toomim (brainworkshop.sourceforge.net, maintained fork). Cerevana shares no code with Brain Workshop (GPL-2.0) and credits it as protocol source and inspiration.

Quad Box — soamsy

The Quad Box exercise is soamsy/quad-box (MIT), vendored into Cerevana and restyled to match the app — game mechanics unchanged (playable original). Built by soamsy, the same developer whose Syllogimous-v3 fork Cerevana’s RRT grew from.
